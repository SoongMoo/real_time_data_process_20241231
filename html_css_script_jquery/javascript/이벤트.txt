1. <input type="text">, <input type="password">, <input type="email">, <input type="search">
이러한 텍스트 기반 input 요소들은 기본적인 텍스트 입력과 관련된 여러 이벤트를 사용할 수 있습니다.

input: 사용자가 값을 입력할 때마다 발생. (최신 입력값을 실시간으로 받아올 수 있음)
change: 입력 값이 변경되고 포커스를 잃었을 때 발생.
focus: 입력란에 포커스가 갔을 때 발생.
blur: 입력란에서 포커스가 빠져나갈 때 발생.
keydown, keyup, keypress: 키보드 키를 누를 때, 떼었을 때 발생 (예: 글자 입력 시).
input: 실시간으로 텍스트가 입력될 때마다 발생 (반복적으로 실행).
cut, copy, paste: 클립보드와 관련된 작업이 있을 때 발생 (예: 텍스트를 자르거나 복사, 붙여넣기).

2. <input type="number">
숫자 입력을 위한 input 요소는 주로 숫자에 대한 이벤트를 처리할 수 있습니다.

input: 실시간으로 값이 변경될 때마다 발생.
change: 값이 변경되고, 입력이 끝났을 때 발생.
focus, blur: 입력란에 포커스가 갔을 때와 빠져나갈 때 발생.
keydown, keyup, keypress: 키 입력을 감지.

3. <input type="checkbox">
체크박스의 경우 체크 상태 변경과 관련된 이벤트를 다룹니다.

change: 체크박스의 상태가 변경될 때 발생.
focus, blur: 체크박스에 포커스가 갔을 때와 빠져나갈 때 발생.

4. <input type="radio">
라디오 버튼은 서로 다른 버튼들 중 하나를 선택하는 데 사용되므로, 상태 변경을 처리하는 이벤트가 중요합니다.

change: 라디오 버튼의 선택 상태가 바뀔 때 발생.
focus, blur: 라디오 버튼에 포커스가 갔을 때와 빠져나갈 때 발생.

5. <input type="file">
파일 업로드를 위한 input 요소입니다.

change: 파일이 선택될 때 발생.
focus, blur: 파일 입력란에 포커스가 갔을 때와 빠져나갈 때 발생.

6. <input type="date">, <input type="time">, <input type="month">, <input type="week">
날짜와 시간 입력을 위한 input 요소는 관련된 값이 변경될 때 발생하는 이벤트를 다룹니다.

change: 날짜나 시간이 선택될 때 발생.
focus, blur: 날짜/시간 입력란에 포커스가 갔을 때와 빠져나갈 때 발생.

7. <input type="color">
색상 선택을 위한 input 요소입니다.

input: 색상이 변경될 때마다 발생.
change: 색상이 선택된 후 발생.
focus, blur: 색상 입력란에 포커스가 갔을 때와 빠져나갈 때 발생.

8. <input type="tel">, <input type="url">
전화번호나 URL 입력을 위한 input 요소입니다.

input: 사용자가 입력할 때마다 발생.
change: 입력이 끝난 후 값이 변경되었을 때 발생.
focus, blur: 포커스와 관련된 이벤트.

9. <input type="hidden">
이 입력 요소는 사용자에게 보이지 않지만, 폼 전송 시 값을 전송할 수 있도록 사용됩니다.

change: 값이 변경될 때 발생.
